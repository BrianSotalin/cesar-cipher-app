# César Cipher App 🔐

Una aplicación web interactiva que permite cifrar y descifrar mensajes utilizando el clásico método de **Cifrado César**. Ideal para aprender sobre criptografía básica o enviar mensajes "secretos" de forma divertida.

## 🚀 Características
- **Cifrado en tiempo real**: Codifica tus mensajes instantáneamente.
- **Descifrado**: Recupera el mensaje original conociendo el número de desplazamiento.
- **Desplazamiento ajustable**: Permite elegir el valor de la clave (offset).
- **Interfaz intuitiva**: Diseño limpio y fácil de usar.

## 🛠️ Tecnologías utilizadas
- **HTML5**: Estructura de la aplicación.
- **CSS3**: Estilos y diseño responsivo.
- **JavaScript (Vanilla)**: Lógica de cifrado y manipulación del DOM.

## 📋 Cómo funciona
El cifrado César es una técnica de sustitución en la que cada letra del texto es reemplazada por otra que se encuentra un número fijo de posiciones más adelante en el alfabeto. Por ejemplo, con un desplazamiento de 3:
- La `A` se convierte en `D`.
- La `B` se convierte en `E`.
